### 4. Root cause analysis
- Khi người dùng nhập một **coupon** không có giá trị phần trăm hợp lệ, backend trả về đối tượng `coupon` với trường `value` **null** (hoặc không phải kiểu số).  
- Trong `Cart.jsx` (dòng 50) code thực hiện:  

  ```tsx

      ? `${coupon.value.toFixed(2)}% off your order`
      : `${coupon.value}$ off your order`;
  ```

- Vì `coupon.value` là `null`, việc gọi `toFixed` sinh ra **TypeError**, khiến toàn bộ trang bị treo (crash).  

- Các coupon “SAVE10” trả về giá trị số hợp lệ, vì thế không gây lỗi.

### 7. Suggested fix

```tsx
// src/pages/Cart.jsx (around line 48‑55)

const discountText = (() => {
  if (!coupon) return '';
  // Ensure value is a number before calling toFixed
  const numericValue = Number(coupon.value);
  if (coupon.type === 'percentage') {
    // Guard against NaN / null
    const percent = isNaN(numericValue) ? 0 : numericValue;
    return `${percent.toFixed(2)}% off your order`;
  }
  const amount = isNaN(numericValue) ? 0 : numericValue;
  return `${amount}$ off your order`;
})();
```
